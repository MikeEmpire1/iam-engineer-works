# 06 — IAM Incident Response

## Status

**Not yet implemented** — scenario template is in place.

## Planned Scope

- Define IAM incident severity levels and response playbooks
- Detect and respond to suspicious IAM activity (unusual API calls, impossible travel, privilege escalation)
- Automate containment: disable compromised IAM Identity Center users, revoke sessions, detach policies
- Forensic analysis using CloudTrail, GuardDuty, and Security Hub
- Post-incident remediation: root cause analysis, control improvements, lessons learned
