# 00 — Company Context

Background materials that define InnoGrid Solutions as an organization, used as the foundation for all IAM scenarios.

## Files

| File | Description |
|---|---|
| `company-profile.md` | Company overview: industry, size, service lines, regulatory compliance, tooling stack |
| `org-chart.md` | Organizational hierarchy: C-suite through individual contributors, ~40 named positions |
| `personnel-roster.md` | 39 employees with names, titles, departments, managers, levels, identity sources, and statuses. Includes terminated employees and staff on leave for lifecycle scenarios |
| `aws-account-structure.md` | 5 AWS accounts across 4 OUs: Management, Security, Production, Nonproduction, Sandbox. Includes SCP guardrails, access model, and break-glass procedure |
| `scp-implementation.md` | Terraform, AWS CLI, and Console deployment for 5 OU-specific Service Boundary SCPs that enforce the Key Services column |
